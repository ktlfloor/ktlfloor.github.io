PRO III

PRO III is a high performance single layer batten system, installed over uneven concrete using different leveling material depending on required construction height. Construction height including 22mm parquet flooring board, without additional packing, is 76mm. Comply with EN 14904, class A4 standard.

Components:
- 22x130mm solid hardwood flooring boards
- Machine nails
- PRO III battens
- Packing
- Moisture barrier
- Concrete

KTL PRO III

  • Single batten system with high shock absorbency and resilience.
  • Comply with EN 14904 - Class A4 standard.
  • Installed over uneven concrete, this is our unique, fast to install and most cost effective batten system. Different leveling methods depending on construction heights.
  • For minor adjustment over partly leveled concrete use bitumen paper. The construction height, excluding packing, is 76mm.
  • Using the unique wedge system and up to 2 raiser blocks, the floor system can cover construction heights from 84 to 144mm.
  • Build-in deformation control of elastic shock pads and heavy duty loading capacity.
Components:
  • 22x130x2180mm solid hardwood flooring boards
  • 2.2x45mm machine nails
  • PRO III battens, 45x58x3750mm c/c 363.3mm or 311.4mm, with shock pads every 750mm c/c
  • Packing, e.g. KTL wedges, yellow wedges, plywood, bitumen paper, only 3.7 packings/m2
  • Min 0.20mm PE-folio (over concrete)
  • Concrete
