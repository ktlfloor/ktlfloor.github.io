Batten system

Solid 20mm plank and 22mm parquet flooring boards can be installed as selfsupported floors by nailing onto leveled battens over concrete or wooden based subfloors.

Components;
- 20x130/155/180mm plank or 22x130mm parquet flooring boards.
- 2.2x45mm machine nails (secretly nailed)
- Battens, made from LVL material.
- Packing, e.g. stable cradles, wedges, plywood block and bitumen paper.
- Moisture barrier - min 0.20mm PE folio (over concrete)

Product Category : Installation systems

  • Description
  • Gallery
  • Inquire Me

  • ﻿Solid 20mm plank or 22mm parquet flooring boards are nailed to leveled battens over concrete or wooden based subfloor, using machine nails, alternatively screws after pre-drilling.

  • Battens are made from Laminated Veneer Lumber, alternatively solid timber, in a moisture level equivalent to the flooring supplied.

  • Batten distance is defined based on application and required load capacity of the floor. See table in Specification document.

  • Leveled to +/- 2mm on 2m, using packing made from solid material, e.g. stable cradles, wedges, plywood blocks, bitumen paper.

  • Moisture barrier on concrete slabs, min. 0.20mm PE-foil.

  • If required insulation can be installed in the void below the flooring boards. Pay attention to the possible maximum thickness of 50mm!
